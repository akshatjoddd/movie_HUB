## 🎬 Movie Explorer (JavaScript Project)
Name - CINEVERSE

**Movie Explorer** is a dynamic web application built using **vanilla JavaScript, HTML, and CSS** that allows users to search and explore movies in real time and manage a personalized watchlist.

The application integrates with movie APIs such as **TMDB / OMDb / Watchmode** to fetch live movie data including posters, ratings, and details, providing an interactive browsing experience.

---

## 🚀 Key Features

* 🔍 **Real-Time Search**
  Users can search for movies instantly using API calls, with optimized input handling (debouncing).

* 🎞️ **Dynamic Movie Rendering**
  Movies are displayed using **DOM manipulation** (`innerHTML`, event listeners) in a responsive grid layout.

* ⭐ **Watchlist Feature**
  Users can add movies to a watchlist, and the button updates dynamically (e.g., **“Add → Done ☑️”**).

* 📄 **Pagination**
  Navigate through multiple pages of movie results efficiently.

* 💾 **Local Storage Support**
  Watchlist is saved in the browser and persists after refresh.

* 🎨 **UI & Animations**
  Dark-themed interface with hover effects and smooth page load animations.

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript (ES6+)
* **API Integration:** Fetch API (async/await)
* **DOM Handling:** Vanilla JS (`querySelector`, `addEventListener`, `innerHTML`)
